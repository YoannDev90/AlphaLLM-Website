Guide complet pour maîtriser le bot Discord IA

Installation et Configuration

Pour ajouter AlphaLLM à votre serveur Discord, cliquez sur le bouton "Inviter" depuis la page d'accueil. Accordez les permissions nécessaires pour que le bot fonctionne correctement.

Permissions Requises

  • Lire les messages
  • Envoyer des messages
  • Utiliser des commandes slash
  • Gérer les messages (optionnel pour la modération)

Commandes Principales

/chat

Démarrez une conversation avec l'IA. Exemple : /chat message="Bonjour, comment ça va ?"

/image

Créez des images uniques à partir de simples descriptions textuelles.

/search

Trouvez rapidement des informations et des réponses grâce à notre moteur de recherche IA intégré.

Modèles Disponibles

AlphaLLM supporte plusieurs modèles IA. Consultez la page d'accueil pour la liste complète.

FAQ

Le bot ne répond pas ?

Vérifiez les permissions et assurez-vous que le bot est en ligne (voir la page Statut).

Comment changer de modèle ?

Utilisez la commande /model pour sélectionner un modèle spécifique.
