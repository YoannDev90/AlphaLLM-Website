Questions Fréquentes

Pour inviter AlphaLLM sur votre serveur Discord, cliquez simplement sur le bouton "Inviter" en haut de la page d'accueil. Vous serez redirigé vers Discord où vous pourrez sélectionner le serveur et les permissions nécessaires. Assurez-vous d'avoir les droits d'administrateur sur le serveur où vous souhaitez ajouter le bot.

Oui, AlphaLLM est entièrement gratuit à utiliser avec toutes ses fonctionnalités. Nous ne proposons pas de version premium pour le moment. Notre objectif est de rendre l'IA accessible à tous les utilisateurs Discord sans barrière financière.

Après avoir invité le bot, utilisez la commande /setup pour accéder au menu de configuration. Vous pourrez définir le canal de discussion par défaut, les permissions, et d'autres paramètres. Pour une configuration plus avancée, vous pouvez également utiliser notre tableau de bord web en vous connectant avec votre compte Discord.

AlphaLLM propose plusieurs modèles d'IA avancés, dont ChatGPT, Claude, Gemini, Mistral, LLaMA et d'autres. Vous pouvez choisir votre modèle préféré avec la commande /model ou définir un modèle par défaut pour votre serveur dans les paramètres.

Pour générer des images, utilisez la commande /imagine suivie de votre description. Vous pouvez spécifier différents styles et ratios d'aspect. Par exemple : /imagine une forêt enchantée avec des fées lumineuses, style:fantasy ratio:16:9

Vous pouvez limiter l'accès au bot à certains rôles ou canaux spécifiques via la commande /permissions ou dans les paramètres du tableau de bord. Vous pouvez également définir des quotas d'utilisation pour éviter les abus.

Vous pouvez signaler les bugs sur notre serveur Discord dans le canal #bug-reports ou en contactant directement notre équipe de support par email. Merci de fournir le plus de détails possibles, comme les étapes pour reproduire le problème, des captures d'écran et le contexte.
